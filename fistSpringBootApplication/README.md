# FirstSpringBootApplication example

STEPS TO RUN THIS APPLICATION.

1. Aplication build in SpringToolSuite(STS) IDE.
2. Create a Project as Spring Starter Project.
3. Add Spring web dependency becz it's a (web application). Give me 1 min to download all dependencies.
4. Create a Plain java class -> Welcome.java -> basic greetmethod().
5. Add Annotations @RestController @GetMapping("/") -> save Run the Main Application.
6. Once Run the application open Any search engine and run on the search bar.  http://localhost:8080/
